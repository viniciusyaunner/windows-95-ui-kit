About

I’m a back-end programmer with a bit of a front-end too, I’m interested in FOSS culture, network infrastructure, natural language processing, minimalist art, futurism, vaporwave and art nouveau.

I graduated in the technical course and then I started to work in the job market as an intern in mobile applications, being hired as a back end programmer. I have been in the market for 3 years.

I worked first as a mobile programmer, however, I received an opportunity to work on a chatbot project and ended up migrating my career to the back end, since then I have worked with Ruby, Python, NodeJS and their respective web frameworks RubyOnRails, Django, Flask and Express.

Since then I've been using Clean Code methodology, working in a dedicated team, mainly because I'm used to teams that use SCRUM methodology and versioning using Git.

As relevant projects that I had, I participated in a Scientific Initiation project together with a Faculty to create a chatbot that analyzed texts and trained itself if it did not know how to interpret what was said. I actively participate in the open source community, especially the Arch Linux community. And I also work as a translator as a hobby.
